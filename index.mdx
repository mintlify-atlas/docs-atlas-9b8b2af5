ALESPI Solutions Web is a corporate landing page for ALESPI Solutions S.A. de C.V., an HVAC engineering company based in Puebla, Mexico. The site presents the company's services, technical expertise, certifications, institutional client portfolio, and contact information in a modern, fully responsive interface built with HTML, Tailwind CSS v3, and vanilla JavaScript.

The project uses a modular partial-loading architecture: `index.html` serves as the application shell, and each section (hero, services, expertise, credentials, contact) lives in its own HTML file under `src/`, loaded dynamically at runtime via `fetch()`. Navigation between sections is handled by a tab system managed in `script.js`.

## Getting started

<Steps>
  <Step title="Clone the repository">
    Download the project source code to your local machine.

    ```bash
    git clone https://github.com/Pollo184/ALESPI-Solutions-Web.git
    cd ALESPI-Solutions-Web
    ```
  </Step>
  <Step title="Install dependencies">
    Install the build tools — Tailwind CSS, PostCSS, and live-server.

    ```bash
    npm install
    ```
  </Step>
  <Step title="Build styles and start the server">
    Compile the CSS and launch the local development server.

    ```bash
    npm run preview
    ```

    The site is available at `http://localhost:8080`.
  </Step>
</Steps>

<Note>
  The site uses `fetch()` to load HTML partials, so it must be served through an HTTP server. Opening `index.html` directly from the filesystem will not work.
</Note>
