## Contact form

The form carries the ID `#formularioContacto` and is handled by `initializeContactForm()` in `script.js`.

### Form fields

All four fields are required:

| Field ID | Label | Type |
|----------|-------|------|
| `nombre` | Nombre Completo | `text` |
| `email` | Email | `email` |
| `asunto` | Asunto | `text` |
| `mensaje` | Mensaje | `textarea` (5 rows) |

```html contacto.html
<form
  class="contacto-form bg-white p-6 sm:p-8 md:p-10 rounded-2xl shadow-2xl border-l-4 border-primary-500"
  id="formularioContacto"
>
  <!-- Nombre Completo -->
  <input type="text" id="nombre" name="nombre" required placeholder="Tu nombre" />

  <!-- Email -->
  <input type="email" id="email" name="email" required placeholder="tu@email.com" />

  <!-- Asunto -->
  <input type="text" id="asunto" name="asunto" required placeholder="Asunto del mensaje" />

  <!-- Mensaje -->
  <textarea id="mensaje" name="mensaje" rows="5" required placeholder="Tu mensaje aquí..."></textarea>

  <button type="submit" class="btn-primary btn-primary-dark w-full">
    Enviar Mensaje
  </button>
</form>
```

### Form validation logic

`initializeContactForm()` attaches a `submit` listener and performs two validation checks before accepting the form:

```js script.js
formularioContacto.addEventListener('submit', (e) => {
  e.preventDefault();

  const nombre  = document.getElementById('nombre').value.trim();
  const email   = document.getElementById('email').value.trim();
  const asunto  = document.getElementById('asunto').value.trim();
  const mensaje = document.getElementById('mensaje').value.trim();

  // Check 1: all fields must be non-empty
  if (!nombre || !email || !asunto || !mensaje) {
    mostrarNotificacion('Por favor, completa todos los campos.', 'error');
    return;
  }

  // Check 2: email must match basic RFC pattern
  const regexEmail = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
  if (!regexEmail.test(email)) {
    mostrarNotificacion('Por favor, ingresa un email valido.', 'error');
    return;
  }

  console.log('Formulario enviado:', { nombre, email, asunto, mensaje });
  mostrarNotificacion('Mensaje enviado exitosamente. Te contactaremos pronto.', 'success');
  formularioContacto.reset();
});
```

### Notification system

Validation outcomes and submission confirmation are displayed via `mostrarNotificacion(mensaje, tipo)`. Notifications appear as fixed-position toasts in the top-right corner (full-width on mobile) and auto-dismiss after 5 seconds with a slide-out animation.

| `tipo` value | Background color | Use case |
|---|---|---|
| `'error'` | `#f44336` (red) | Empty fields or invalid email |
| `'success'` | `#4caf50` (green) | Successful form submission |
| `'info'` | `#2196f3` (blue) | General information |

<Warning>
  The form currently performs **frontend-only validation**. On a valid submission, the form data is logged to the browser console with `console.log(...)` but is not sent anywhere. You must integrate a backend endpoint or a serverless function (for example, a Netlify Function, Vercel serverless route, or a third-party form service such as Formspree) to actually deliver messages.
</Warning>
