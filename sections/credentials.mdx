### Animated number counting

When the stats section scrolls into view, `initializeStatsObserver()` in `script.js` triggers `animarNumero()` on each `.stat-number` element. The function counts up from zero to the target value over 2 seconds:

```js script.js
function animarNumero(element) {
  const numero = element.textContent;
  const valorNumerico = parseInt(numero) || 0;
  const duracion = 2000; // 2 segundos
  const incremento = valorNumerico / (duracion / 50);

  let valorActual = 0;
  const intervalo = setInterval(() => {
    valorActual += incremento;
    if (valorActual >= valorNumerico) {
      element.textContent = numero;   // restore original string, e.g. "100+"
      clearInterval(intervalo);
    } else {
      element.textContent = Math.floor(valorActual) + '+';
    }
  }, 50);
}
```

<Note>
  The observer uses a `threshold: 0.5` — the animation fires only when at least 50% of the stats section is visible in the viewport. Each element is unobserved after its animation runs so it does not repeat on re-scroll.
</Note>

The logo grid displays five client logos in a `grid-cols-2 md:grid-cols-3` layout. Each logo sits in a `h-36 rounded-3xl bg-light shadow-md` card:

| Institution | Image file |
|-------------|-----------|
| Guardia Nacional | `images/guadria_nacional.svg` |
| ANAM | `images/anam.png` |
| IMSS | `images/imss.png` |
| ISSSTE | `images/issste.png` |
| SEDENA | `images/sedena.png` |

A sixth slot is reserved as a dashed-border placeholder for an additional institutional client.

<Tip>
  To add a new client logo, replace the placeholder div with an `<img>` tag inside the existing `h-36 rounded-3xl bg-light shadow-md flex items-center justify-center p-4` wrapper and add the institution name to the `<ul>` in the dark left panel.
</Tip>
