The hero section is the first thing visitors see when they land on the ALESPI Solutions site. It lives in `src/hero.html` and is loaded alongside `src/valores.html` into the `inicio` tab by `loadPartials()` in `script.js`.

## Purpose

The hero section serves three goals:

- Communicate the company's core value proposition at a glance
- Surface key trust indicators (project count, years of experience, sector focus)
- Guide visitors toward the two most important next steps via CTA buttons

## Content elements

### Headline and subheadline

The headline reads:

> Soluciones HVAC con una presencia de marca **solida y profesional**

The word "solida y profesional" is wrapped in a `hero-highlight` span that applies the brand gradient. Below the headline, the subheadline describes ALESPI's scope:

> Diseno, instalacion y mantenimiento de sistemas de climatizacion para sector institucional, comercial e industrial, con enfoque en cumplimiento tecnico, continuidad operativa y confianza de largo plazo.

A pill-shaped badge above the headline reads:

> Ingenieria HVAC con identidad tecnica y ejecucion confiable

### CTA buttons

Two call-to-action buttons sit below the subheadline. Each button uses the `data-open-tab` attribute so that `initializeTabs()` in `script.js` intercepts the click, prevents default navigation, and programmatically activates the target tab.

<CardGroup cols={2}>
  <Card title="Solicitar cotizacion" icon="arrow-right">
    Primary button. Links to the `contacto` tab. Uses `btn-primary btn-primary-light` classes with a hover lift and ripple effect.
  </Card>
  <Card title="Conocer servicios" icon="arrow-right">
    Secondary button. Links to the `servicios` tab. Outlined style — white border that fills on hover.
  </Card>
</CardGroup>

The HTML for a CTA button with `data-open-tab`:

```html hero.html
<a
  href="#contacto"
  data-open-tab="contacto"
  class="btn-primary btn-primary-light inline-flex w-full sm:w-auto justify-center
         hover:shadow-2xl hover:-translate-y-2 transition-all duration-300 group btn-ripple relative"
>
  <span class="flex items-center justify-center gap-2">
    Solicitar cotizacion
    <i class="fas fa-arrow-right group-hover:translate-x-2 transition-transform duration-300"></i>
  </span>
</a>
```

<Note>
  The `href` attribute is present for accessibility and progressive enhancement, but `initializeTabs()` calls `event.preventDefault()` so the page never actually navigates to an anchor — it switches tabs instead.
</Note>

### How tab switching works

`initializeTabs()` queries all `[data-open-tab]` elements and attaches a click listener to each:

```js script.js
tabLinks.forEach((link) => {
  link.addEventListener('click', function(event) {
    event.preventDefault();
    activateTab(this.getAttribute('data-open-tab'));
    navElement?.classList.remove('expanded');
  });
});
```

`activateTab(tabName)` hides all `.tab-content` elements, removes `tab-active` from all nav buttons, then reveals the target content panel and restarts its CSS animations.

## Animated background

The hero background uses layered CSS effects defined directly in `src/hero.html`:

<AccordionGroup>
  <Accordion title="Radial gradients">
    Two `radial-gradient` overlays are applied via an inline `bg-[...]` utility on the section's first child div — one at the top-left corner and one at the bottom-right, both using semi-transparent brand blue tones.
  </Accordion>
  <Accordion title="Floating circles">
    Three large white circles with low opacity (`opacity-10`) use custom Tailwind animations: `animate-float-1`, `animate-float-2`, and `animate-float-3`. These are defined in `tailwind.config.js` and produce a gentle vertical bob.
  </Accordion>
  <Accordion title="Pulsing ring and bouncing dot">
    A 128 × 128 px outlined circle uses `animate-pulse`, and a small dot uses `animate-bounce-custom`. Both are hidden on mobile via `hidden sm:block`.
  </Accordion>
</AccordionGroup>

## File location

The hero markup lives in:

```
src/hero.html
```

It is injected into `index.html` via the `data-include` mechanism:

```html index.html
<div data-include="src/hero.html"></div>
```

The `inicio` tab also loads `src/valores.html` (Mision, Vision, Fortaleza) in the same panel.
